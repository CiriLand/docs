# Regole del Server CiriLand

Per garantire un'esperienza di gioco piacevole per tutti, abbiamo stabilito alcune regole fondamentali che ogni giocatore deve rispettare. Il mancato rispetto di queste regole può comportare avvertimenti, restrizioni temporanee o, nei casi più gravi, ban permanenti.

## Regole Generali

### 1. Rispetto verso gli altri giocatori

- Non insultare, molestare o discriminare altri giocatori
- Evita linguaggio offensivo o inappropriato in chat
- Rispetta le decisioni dello staff
- Non provocare intenzionalmente altri giocatori

### 2. Gioco leale

- Non utilizzare hack, mod non autorizzate o exploit
- Non abusare di bug o glitch del gioco
- Non utilizzare programmi esterni per ottenere vantaggi
- Non condividere account o prestare le tue credenziali

### 3. Costruzioni e territorio

- Rispetta le costruzioni degli altri giocatori
- Non modificare o distruggere costruzioni altrui senza permesso
- Mantieni una distanza ragionevole dalle costruzioni di altri giocatori
- Non rivendicare territori eccessivamente grandi senza utilizzarli

### 4. Economia e commercio

- Non truffare altri giocatori durante gli scambi
- Rispetta gli accordi presi con altri giocatori
- Non abusare del sistema economico del server
- Segnala eventuali problemi economici allo staff

## Regole Specifiche per la Modalità RPG

### 1. Classi e abilità

- Non abusare delle abilità delle classi per danneggiare l'esperienza di gioco altrui
- Non sfruttare combinazioni di abilità per ottenere vantaggi sleali
- Segnala eventuali problemi di bilanciamento allo staff

### 2. Dungeon e boss

- Non bloccare l'accesso ai dungeon ad altri giocatori
- Rispetta i turni durante gli eventi di boss
- Non rubare il loot di altri giocatori nei dungeon
- Collabora con gli altri durante le attività di gruppo

### 3. PvP

- Il PvP è consentito solo nelle aree designate
- Non attaccare ripetutamente lo stesso giocatore (no spawn killing)
- Non utilizzare abilità di classe per intrappolare altri giocatori
- Rispetta le regole specifiche degli eventi PvP

## Sanzioni

Le violazioni delle regole possono comportare le seguenti sanzioni, applicate a discrezione dello staff:

1. **Avvertimento**: Per infrazioni minori o prime violazioni
2. **Mute temporaneo**: Impossibilità di utilizzare la chat per un periodo di tempo
3. **Jail**: Confinamento in un'area specifica del server
4. **Ban temporaneo**: Esclusione dal server per un periodo di tempo
5. **Ban permanente**: Esclusione definitiva dal server

La gravità della sanzione dipenderà dalla natura dell'infrazione, dalla sua frequenza e dall'atteggiamento del giocatore.

## Segnalazioni

Se noti comportamenti che violano le regole, puoi segnalarli in diversi modi:

- Comando in-game: `/report <giocatore> <motivo>`
- Discord: Canale #segnalazioni
- Email: report@ciriland.com

Fornisci sempre prove concrete (screenshot, registrazioni, timestamp) quando possibile.

## Appelli

Se ritieni che una sanzione sia stata applicata ingiustamente, puoi fare appello:

1. Attendi almeno 24 ore dopo la sanzione
2. Invia un'email a appeals@ciriland.com con:
   - Il tuo nome utente
   - La data della sanzione
   - Una spiegazione dettagliata del tuo punto di vista
   - Eventuali prove a tuo favore

## Modifiche alle Regole

Lo staff si riserva il diritto di modificare queste regole in qualsiasi momento. Le modifiche verranno comunicate attraverso:

- Annunci in-game
- Server Discord
- Questa pagina di documentazione

È responsabilità dei giocatori rimanere aggiornati sulle regole del server.
